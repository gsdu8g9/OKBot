# OKBot

OKBot - Позволит вам оставаться всегда быть онлайн в социальной сети Одноклассники.

## Installation

1. Install Node.js 
2. Clone this project
3. Edit index.js file

## Usage
Startup command line or terminal and write next command: <br>
  `npm install` <br>
  `node index.js` <br>
